Tile Flip

The memory tile flip game is a matching challenge where a set of tiles, each with a hidden image, are laid face-down in a grid. Every image appears on exactly two tiles. Players take turns flipping over two tiles: if the images match, the player keeps the pair; if not, the tiles are turned face-down again. The goal is to remember the positions of images and find all matching pairs before the opponent. The game ends when all pairs are matched, and the player with the most pairs wins.
